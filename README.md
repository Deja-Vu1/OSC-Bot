# OSC-Bot

The best bot ever!


# Running it

1. Get your stuff organized:
``` shell
$ python3 -m venv env
$ source env/bin/activate
(env) $ # You should see the (env)!
```

2. Get your deps installed
``` shell
(env) $ pip install -r requirements.txt
```

3. Run the project
``` shell
(env) $ python bot.py
```
